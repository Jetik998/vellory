# Vellory

**Vellory** — современное веб-приложение для управления задачами с системой аутентификации, загрузкой аватаров и удобным фронтендом.

Проект построен на **FastAPI** (backend) + статическом фронтенде (HTML + vanilla JS), работает в Docker-контейнере с PostgreSQL, Redis и Nginx в качестве reverse-прокси. Поддерживает production-развёртывание с автоматическим деплоем через GitHub Actions.

---

## ✨ Основные возможности

- **Аутентификация и авторизация**

  - Регистрация / вход / выход
  - JWT-токены (`access_token` + `refresh_token`) хранятся в HttpOnly cookies
  - Автоматическое обновление токенов
  - Защита эндпоинтов через зависимости FastAPI

- **Управление задачами**

  - CRUD-операции (создание, просмотр, редактирование, удаление)
  - Поддержка дедлайнов (В разработке)
  - Отображение всех задач пользователя в реальном времени

- **Аватары пользователей**

  - Загрузка изображения
  - Автоматическая конвертация в WebP
  - Статическая раздача через Nginx (`/avatars/`)

- **Дополнительно**
  - Rate limiting (20 запросов / 30 сек)
  - OpenAPI-документация (`/docs`, `/redoc`)
  - Health-check, metrics, debug-эндпоинты
  - Полноценный CI/CD (GitHub Actions → GitLab Registry → сервер)

## 🛠 Технологический стек

| Слой             | Технологии                                                                              |
| ---------------- | --------------------------------------------------------------------------------------- |
| **Backend**      | FastAPI 0.121, SQLAlchemy 2.0, Alembic, Pydantic 2, PyJWT, argon2-cffi, fastapi-limiter |
| **БД**           | PostgreSQL 18, Redis 8                                                                  |
| **Frontend**     | HTML + CSS + Vanilla JavaScript                                                         |
| **Web-сервер**   | Nginx (SSL, OCSP Stapling, gzip, proxy)                                                 |
| **Контейнеры**   | Docker + docker-compose                                                                 |
| **CI/CD**        | GitHub Actions + GitLab Container Registry                                              |
| **Тестирование** | pytest + schemathesis                                                                   |

## 🚀 Быстрый запуск (локально)

### 1. Клонирование

```bash
git clone https://github.com/ваш-юзер/Vellory.git
cd Vellory
```

### 2. Настройка окружения

```bash
cp .env.example .env
```

### 3. Запуск через Docker

```bash
docker compose up -d --build
```

## 🌐 Автоматический деплой

Автоматический деплой настроен через **GitHub Actions**.

Процесс запускается при каждом пуше в ветку `main`:

1. **Пуш** в ветку `main`
2. **Сборка** Docker-образов `fastapi` и `nginx`
3. **Пуш** образов в **GitLab Container Registry**
4. **Копирование** файла `compose.yaml` на сервер
5. Выполнение команды:

```bash
docker compose pull && docker compose up -d
```

# 🧪 Тестирование

## 📋 Запуск тестов


```bash
pytest tests/schema/
```
